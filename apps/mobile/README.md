# Pursuit App

A React Native application for the Pursuit project.

## Getting Started

### Prerequisites

- Node.js
- React Native CLI
- iOS Simulator (for iOS development)
- Android Studio (for Android development)

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. For iOS:
   ```bash
   npx react-native run-ios
   ```

3. For Android:
   ```bash
   npx react-native run-android
   ```

## Project Structure

- `app/` - Main application screens
- `assets/` - Images and other static assets
- `App.tsx` - Root application component

## Development

This app is built with React Native and TypeScript.
